# cell_mesh_2
A 3D model of cells growing

## Build instructions
Dependencies:
- git
- cmake
- yaml-cpp >= 0.6

```
git clone https://github.com/robertvi/cell_mesh_2.git
cd cell_mesh_2
mkdir build
cd build
cmake ..
make
./CellMesh2
```
